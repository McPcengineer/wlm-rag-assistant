# wlm-rag-assistant# WLM RAG Assistant (Blue Yonder Warehouse Labor Management)

A Retrieval-Augmented Generation (RAG) system designed to answer operational,
technical, and configuration questions about **Blue Yonder's WLM system**
using internal documentation.

This project uses:
- Python
- LangChain
- OpenAI / BGE Embeddings
- ChromaDB / Pinecone
- FastAPI
- HTML/JS Frontend
- Docker
